# Creating your First Project

> A beginners guide to creating your first project in the s&box editor

# Finding the Editor

To access the s&box editor, you can sort by Tools in your Steam Library, or just search for it directly:[![](https://cdn.sbox.game/upload/b/3b1698af/fd29/449d/b8c8/8736453bbd1a.png)](https://cdn.sbox.game/upload/b/3b1698af/fd29/449d/b8c8/8736453bbd1a.png)

# Using the Launcher

Once you open the s&box editor, you will first be met with the launcher:[![](https://cdn.sbox.game/upload/b/b6153f27/8be8/4438/9c49/bcf466dd3a60.png)](https://cdn.sbox.game/upload/b/b6153f27/8be8/4438/9c49/bcf466dd3a60.png)Here you can create a new project, load an existing project, or select a Sample Project.  
  
There are also some helpful resources on the sidebar containing links to the official Documentation, Workshop, API Reference, and more!

# Creating your First Project

Once you click the "New Project" button you'll see a popup where you can choose a starting template for your project, and the name/path:[![](https://cdn.sbox.game/upload/b/88f409d0/ab4a/4640/a991/46589c249bb8.png)](https://cdn.sbox.game/upload/b/88f409d0/ab4a/4640/a991/46589c249bb8.png)The template descriptions are self-explanatory, So make sure you pick the one that best suits the project that you're trying to make. I'll be making an empty Game project for the sake of this tutorial.  
  
The Title and Ident can be changed at any time, so don't worry if you use a placeholder name for now.

# Testing your Project

Once you press Create, the engine will load the project and present you with the editor.  
  
From here you can press the Play Button at the top (or use the F5 shortcut) to enter and exit Play Mode. This is how you can test your Game/Map/Addon exactly as it would play out when published to s&box.[](https://cdn.sbox.game/upload/b/0e2c4723/eb49/4e3d/ae48/f7d232912118.mp4)You can click on the objects in the scene to Inspect them and move/scale/rotate them around. Try messing about with the given scene and get comfortable with the workflow (like duplicating some of the existing objects, re-colouring them, dragging new objects into your scene via the Cloud Browser, ect)
